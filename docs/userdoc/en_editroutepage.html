The Route Editor

You will reach the route editor with the  button from the navigation page .



Buttons

Icon Name Function

ZoomIn zoom in

ZoomOut zoom out

NavAdd add a new point to the route
the chart center (cross) will be added after the point that is currently marked red

NavDelete delete the point that is currently marked red

NavToCenter move the currently marked red point to the center of the chart (cross)

NavGoto start navigation to the currently marked red point

MOB man over board(see main page)

Cancel back to the navigation page

At this page you can create an edit a route.

At each point in time there is a currently selected point within the route. It is marked red on the chart and marked grey in the list on the left side. You can change the selected point by clicking on a different point on the chart or in the list on the left side.
If you are editing the currently active route, the route info at the left side will have a red border (otherwise green).
If you leave the editor the display will go back to the active route.

All changes will become effective immediately - there is no fallback.

At the left side you have the list of all points in the route with courses and distances. In the upper display you have the overall route length and the estimated ETA (if the route is active).
By clicking on an already selected waypoint in the list you can open an edit dialog to change the name or position.
Clicking on the route info on the left side will move you to the route list. At this page you can delete all points, change the name, invert or load a new route.

Clicking on the lower left panel will bring up some additional buttons.

Using those buttons you can move between the points in the route or you can edit the selected point.
Clicking on the lower right will center the chart to the boat position.
