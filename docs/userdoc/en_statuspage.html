The Status Page

From the main page you get here by clicking the  button. You will see some server internal status information.


Buttons

Icon Name Function

StatusWpa go to wifi configuration to set up an external wifi connection
(not on android, only visible if configured)

StatusAddresses go to the  list of server addresses. QR codes are displayed, allowing devices to connect by scan

StatusAndroid go to android settings (only android in normal mode)

StatusShutdown gracefully shutdown the server (not android). A confirmation dialog will be displayed. After the shutdown has started you should wait for the headline to turn red.

MainInfo license and privacy information

MOB man over board (see main page)

Cancel return to previous page

Status information of the several server parts are shown. You can check connections and verify if NMEA data are received. This page is important for trouble shooting.
The page updates automatically.
